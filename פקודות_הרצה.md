# פקודות הרצה – שלב אחר שלב

**להריץ מכאן (Run):** לחץ **Ctrl+Shift+P** → הקלד **Run Task** → בחר את השלב (שלב 1, שלב 2, ...).

---

## שלב 1: מעבר לתיקיית הפרויקט

**▶ הרץ מכאן:** Ctrl+Shift+P → Run Task → **שלב 1: מעבר לתיקייה**

```powershell
cd c:\Users\USER\Downloads\mytest1
```

---

## שלב 2: הפעלת הסביבה הווירטואלית (אם יש)

**▶ הרץ מכאן:** Run Task → **שלב 2: הפעלת venv**

```powershell
.\.venv\Scripts\activate
```

(אם אין לך `.venv` – דלג לשלב 4.)

---

## שלב 3: התקנת תלויות (פעם ראשונה בלבד)

**▶ הרץ מכאן:** Run Task → **שלב 3: התקנת תלויות**

```powershell
pip install -r requirements.txt
```

---

## שלב 4: הרצת האפליקציה

**▶ הרץ מכאן:** Run Task → **שלב 4: הרצת האפליקציה**

```powershell
python app.py
```

---

## שלב 5: פתיחת האתר

**▶ הרץ מכאן:** Run Task → **שלב 5: פתיחת האתר בדפדפן**

בדפדפן: **http://localhost:8000**

---

## סיכום (העתק והרץ לפי סדר)

```powershell
cd c:\Users\USER\Downloads\mytest1
.\.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
